📤 파일 업로드 방법:

  1. Google Drive 폴더 접속
    drive.google.com
  2. 수료증 폴더로 이동
    공유된 "수료증" 폴더를 클릭하세요
  3. 파일 업로드
    PDF 파일들을 드래그 앤 드롭하거나
    "새로 만들기" → "파일 업로드" 클릭

📋 파일명 규칙 (필수):

  • 형식: 성명_생년월일.pdf
  • 예시: 홍길동_1990-01-01.pdf
  • 주의: 파일명은 정확히 일치해야 다운로드됩니다

✅ Google Drive의 장점:

  • 무료 15GB 저장 공간
  • 익숙한 인터페이스
  • 드래그 앤 드롭으로 간편 업로드
  • 파일 미리보기 가능

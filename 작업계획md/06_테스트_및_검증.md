# 06. 테스트 및 검증

## 목표
전체 시스템의 정상 동작 확인

## 테스트 항목

### 6.1 서버 테스트
- [ ] 서버 정상 실행 (포트 3000)
- [ ] 정적 파일 서빙 확인
- [ ] API 응답 확인

### 6.2 사용자 기능 테스트

#### 정상 케이스
- [ ] 수료자 정보 입력 → 수료증 다운로드 성공
- [ ] 다운로드된 PDF 파일 정상 확인

#### 에러 케이스
- [ ] 존재하지 않는 수료자 → 에러 메시지 표시
- [ ] 수료자는 있지만 파일 없음 → 관리자 문의 메시지
- [ ] 빈 입력값 → 필수 입력 안내

### 6.3 관리자 기능 테스트

#### 로그인
- [ ] 올바른 비밀번호 → 대시보드 이동
- [ ] 잘못된 비밀번호 → 에러 메시지
- [ ] 미인증 상태로 대시보드 접근 → 로그인 페이지 이동

#### 수료증 업로드
- [ ] 단일 파일 업로드
- [ ] 다중 파일 업로드 (10개)
- [ ] 드래그 앤 드롭 업로드
- [ ] 150개 초과 시 제한 메시지

#### 수료증 목록 관리
- [ ] 목록 정상 표시
- [ ] 검색 필터링 동작
- [ ] 개별 파일 삭제
- [ ] 새로고침 버튼 동작

#### 수료자 리스트 관리
- [ ] JSON 다운로드
- [ ] JSON 업로드 및 반영
- [ ] 잘못된 JSON 형식 업로드 → 에러 메시지

### 6.4 통합 테스트

#### 시나리오 1: 관리자 워크플로우
1. 관리자 로그인
2. 수료자 리스트 업로드
3. 수료증 PDF 대량 업로드
4. 업로드 확인
5. 로그아웃

#### 시나리오 2: 사용자 워크플로우
1. 메인 페이지 접속
2. 본인 정보 입력
3. 수료증 다운로드
4. 파일 확인

### 6.5 브라우저 호환성 테스트
- [ ] Chrome
- [ ] Edge
- [ ] Firefox
- [ ] Safari (가능시)

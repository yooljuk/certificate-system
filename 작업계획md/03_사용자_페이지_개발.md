# 03. 사용자 페이지 개발

## 목표
수료자가 수료증을 다운로드할 수 있는 메인 페이지 구현

## 작업 항목

### 3.1 HTML (public/index.html)

#### 페이지 구성
- 제목: "2026 탄소중립평가사 수료증 발급"
- 입력 폼:
  - 성명 입력 필드
  - 생년월일 입력 필드 (date 타입)
  - 다운로드 버튼
- 로딩 표시 영역
- 에러/성공 메시지 영역
- 관리자 로그인 링크

### 3.2 CSS (public/css/main.css)

#### 스타일 요소
- 반응형 레이아웃
- 폼 스타일링
- 버튼 스타일
- 메시지 스타일 (에러: 빨강, 성공: 초록)
- 로딩 애니메이션

### 3.3 JavaScript (public/js/main.js)

#### 기능 구현
- 폼 제출 이벤트 처리
- API 호출 (`/api/download-certificate`)
- 응답 타입 확인 (PDF vs JSON)
- PDF 다운로드 처리
- 로딩 상태 관리
- 에러/성공 메시지 표시 (5초 후 자동 숨김)

## 완료 조건
- [ ] index.html 작성 완료
- [ ] main.css 작성 완료
- [ ] main.js 작성 완료
- [ ] 폼 입력 및 제출 테스트
- [ ] 정상 다운로드 테스트
- [ ] 에러 메시지 표시 테스트

# 탄소중립평가사 수료증 발급 시스템 - 작업 계획 개요

## 프로젝트 정보
- **버전**: MVP 1단계 v2
- **작성일**: 2026-01-27

## 시스템 개요
관리자가 미리 제작한 수료증 PDF를 업로드하고, 수료자가 본인 정보를 입력하면 해당 수료증을 자동으로 찾아 다운로드할 수 있는 시스템

## 핵심 기능
- 수료자 정보 기반 검증
- 파일명 기반 수료증 매칭 (`성명_생년월일.pdf`)
- PDF 다운로드
- 관리자 대시보드 (대량 업로드 지원)

## 기술 스택
- **Backend**: Node.js, Express
- **File Upload**: Multer
- **Frontend**: HTML, CSS, JavaScript (Vanilla)
- **Data**: JSON 파일

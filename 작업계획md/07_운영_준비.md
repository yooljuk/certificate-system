# 07. 운영 준비

## 목표
실제 운영을 위한 데이터 준비 및 배포

## 작업 항목

### 7.1 수료증 PDF 준비

#### 파일명 규칙
- 형식: `성명_생년월일.pdf`
- 예시: `홍길동_1990-01-01.pdf`
- 생년월일 형식: `YYYY-MM-DD`

#### 체크리스트
- [ ] 120명 수료증 PDF 제작 완료
- [ ] 파일명 형식 검증 완료
- [ ] 모든 파일명이 students.json과 일치하는지 확인

### 7.2 수료자 데이터 준비

#### students.json 형식
```json
[
  {
    "name": "홍길동",
    "birthDate": "1990-01-01"
  },
  ...
]
```

#### 체크리스트
- [ ] 실제 수료자 120명 데이터 입력
- [ ] 이름 오타 검수
- [ ] 생년월일 형식 검증 (YYYY-MM-DD)
- [ ] PDF 파일명과 데이터 매칭 확인

### 7.3 설정 파일 확인

#### config.json
- [ ] 관리자 비밀번호 변경 (운영용)
- [ ] 비밀번호 보안 수준 확인

### 7.4 데이터 업로드

#### 순서
1. students.json 업로드 (관리자 대시보드)
2. 수료증 PDF 120개 업로드 (드래그 앤 드롭)
3. 업로드 완료 확인
4. 샘플 다운로드 테스트

### 7.5 최종 점검

- [ ] 전체 수료자 대상 다운로드 테스트 (5명 샘플)
- [ ] 에러 케이스 재확인
- [ ] 관리자 기능 재확인

## 운영 체크리스트

### 서버 환경
- [ ] Node.js 설치 확인
- [ ] 포트 3000 사용 가능 확인
- [ ] 방화벽 설정 (필요시)

### 데이터 백업
- [ ] students.json 백업
- [ ] certificates 폴더 백업

### 문서화
- [ ] 관리자 매뉴얼 작성
- [ ] 문제 발생 시 연락처 확보

## 배포 명령어
```bash
# 서버 실행
node server.js

# 백그라운드 실행 (Linux/Mac)
nohup node server.js &

# PM2 사용 시
pm2 start server.js --name certificate-system
```

## 완료 조건
- 모든 수료증 파일 업로드 완료
- 수료자 데이터 등록 완료
- 최종 테스트 통과
- 운영 준비 완료

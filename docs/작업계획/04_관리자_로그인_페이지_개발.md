# 04. 관리자 로그인 페이지 개발

## 목표
관리자 인증을 위한 로그인 페이지 구현

## 작업 항목

### 4.1 HTML (public/admin-login.html)

#### 페이지 구성
- 제목: "관리자 로그인"
- 비밀번호 입력 필드
- 로그인 버튼
- 메인으로 돌아가기 링크
- 에러 메시지 영역

### 4.2 CSS (public/css/admin.css)

#### 스타일 요소
- 로그인 폼 중앙 정렬
- 입력 필드 스타일
- 버튼 스타일
- 에러 메시지 스타일

### 4.3 JavaScript (public/js/admin-login.js)

#### 기능 구현
- 폼 제출 이벤트 처리
- API 호출 (`/api/admin/login`)
- 인증 성공 시:
  - sessionStorage에 인증 상태 저장
  - admin-dashboard.html로 이동
- 인증 실패 시:
  - 에러 메시지 표시

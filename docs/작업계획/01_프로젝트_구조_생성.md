# 01. 프로젝트 구조 생성

## 목표
MVP 1단계 개발을 위한 기본 폴더 및 파일 구조 생성

## 작업 항목

### 1.1 폴더 구조 생성
```
project/
├── public/
│   ├── css/
│   └── js/
├── data/
└── certificates/
```

### 1.2 초기 설정 파일 생성

#### config.json
- 관리자 비밀번호 설정
- 기본값: `admin1234`

#### data/students.json
- 수료자 리스트 초기 데이터
- 형식: `[{ "name": "홍길동", "birthDate": "1990-01-01" }]`

## 참고사항
- certificates 폴더는 업로드된 PDF 파일 저장용
- 파일명 규칙: `성명_생년월일.pdf` (예: `홍길동_1990-01-01.pdf`)

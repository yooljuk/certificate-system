# 02. 서버 개발 (server.js)

## 목표
Node.js + Express 기반 백엔드 서버 구축

## 사전 작업
```bash
npm init -y
npm install express multer
```

## 작업 항목

### 2.1 기본 서버 설정
- Express 앱 초기화
- JSON 파싱 미들웨어
- 정적 파일 서빙 (public 폴더)

### 2.2 파일 업로드 설정 (Multer)
- 저장 경로: `./certificates`
- 파일명: 원본 유지 (한글 지원)
- 제한: 파일당 10MB, 최대 150개

### 2.3 사용자 API

#### POST /api/download-certificate
- 입력: `{ name, birthDate }`
- 처리:
  1. students.json에서 수료자 검증
  2. 파일명 생성: `성명_생년월일.pdf`
  3. 파일 존재 확인 후 다운로드
- 응답: PDF 파일 또는 에러 메시지

### 2.4 관리자 API

#### POST /api/admin/login
- 비밀번호 검증

#### POST /api/admin/upload-certificates
- 수료증 대량 업로드 (최대 150개)

#### GET /api/admin/certificates
- 업로드된 수료증 목록 조회

#### DELETE /api/admin/certificate/:filename
- 수료증 파일 삭제

#### GET /api/admin/students
- 수료자 리스트 조회

#### POST /api/admin/students
- 수료자 리스트 업데이트

# 05. 관리자 대시보드 개발

## 목표
수료증 업로드 및 관리를 위한 관리자 대시보드 구현

## 작업 항목

### 5.1 HTML (public/admin-dashboard.html)

#### 페이지 구성
1. **헤더**
   - 제목: "관리자 대시보드"
   - 로그아웃 버튼

2. **수료증 파일 업로드 섹션**
   - 파일 선택 버튼
   - 드래그 앤 드롭 영역
   - 선택된 파일 목록
   - 업로드 버튼
   - 진행률 표시

3. **업로드된 수료증 목록 섹션**
   - 검색 입력 필드
   - 새로고침 버튼
   - 전체 삭제 버튼
   - 파일 목록 (파일명, 삭제 버튼)
   - 총 개수 표시

4. **수료자 리스트 관리 섹션**
   - JSON 다운로드 버튼
   - JSON 업로드 버튼
   - 등록된 수료자 수 표시

### 5.2 JavaScript (public/js/admin-dashboard.js)

#### 기능 구현

##### 인증 확인
- sessionStorage 체크
- 미인증 시 로그인 페이지로 이동

##### 파일 업로드
- 파일 선택 (multiple)
- 드래그 앤 드롭 이벤트
- PDF 파일만 필터링
- 최대 150개 제한
- 업로드 API 호출

##### 수료증 목록 관리
- 목록 로드 (`/api/admin/certificates`)
- 검색 필터링
- 개별 삭제
- 새로고침

##### 수료자 리스트 관리
- JSON 다운로드
- JSON 업로드 및 검증

##### 로그아웃
- sessionStorage 삭제
- 로그인 페이지로 이동
